# moneyCheck

여러명에서 쓸 수 있는 공유 가계부 프로젝트입니다.



## 주요기능
* SNS로그인
* 가계부 작성
  * 지출내역
  * 수입내역
* 일정 등록
* 엑셀 업로드
  * 지출내역
  * 수입내역
* 그래프 보고서
   * 전체
   * 지출
   * 수입 
* 회원
  * 가계부 공유회원 리스트 
  * 회원 초대하기
  * 초대된 회원들과 채팅
